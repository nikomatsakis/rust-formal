A predicate `R = l(E*)` represents a label `l` applied to the path
`E`.  The labels `l` used in the core of Rust are `init`, `!init`.

    R-Var:
        init(x) in Rs
        ------------------------------------------------------------
        Rs |- x OK

    R-Field:
        Rs |- E OK
        ------------------------------------------------------------
        Rs |- E.f OK

    R-Item:
        ------------------------------------------------------------
        Rs |- i::<Ts> OK

    R-Call:
        Rs |- E OK
        
        Rs_1 |- Es OK (forall)
        ------------------------------------------------------------
        Rs |- E(Es) OK

    R-Deref:
        Rs |- E OK
        ------------------------------------------------------------
        Rs |- *E OK

    R-Let:
        Rs |- E OK
        Rs, init(x) |- E_1 OK
        ------------------------------------------------------------
        Rs |- let x: T = E in E_1 OK

    R-Record:
        Rs |- Es OK (forall)
        ------------------------------------------------------------
        Rs |- { Ms fs: Es } OK

    R-Alt:
        Rs |- E OK
        Rs |- As OK
        ------------------------------------------------------------
        Rs |- alt E { As } OK

    R-Fn:
        Rs |- E OK
        ------------------------------------------------------------
        Rs |- fn(Ds xs: Ts) -> T { E } OK

    R-Assign:
        Rs |- E_1 OK
        Rs |- E_2 OK
        Rs |- E_3 OK
        ------------------------------------------------------------
        Rs |- E_1 = E_2; E_3 OK
