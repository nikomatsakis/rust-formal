## Type Rules ##

    Nsub-equiv:
        ------------------------------------------------------------
        N <= N

    Nsub-const:
        ------------------------------------------------------------
        const <= N

    Tsub-equiv:
        ------------------------------------------------------------
        Env |- T <: T

    Tsub-mut-@:
        N_1 <= N_2
        Env |- T_1 <: T_2
        ------------------------------------------------------------
        Env |- @ N_1 T_1 <: @ N_2 T_2
